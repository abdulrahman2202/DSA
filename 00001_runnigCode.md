
## C Programs:
Compile the program. Assuming you have GCC (via MinGW or similar) installed and on PATH:
```
gcc .\CfileName(ex - .\008_ArrayADT.c ) -o fileName.exe (ex - 008_ArrayADT.exe)
```

Run the compiled program:
```
.\ fileName.exe (ex - .\008_ArrayADT.exe)
```




## C++ Programs:
Compile the program using g++ (MinGW or similar):
```
g++ -o outputName.exe "sourceFile.cpp" (ex - g++ -o 011_Factorial.exe "011_Factorical_loop.c++")
```

Run the compiled program:
```
.\outputName.exe (ex - .\011_Factorial.exe)
```

**Note:** For C++ files, use `g++` instead of `gcc`. The `-o` flag specifies the output executable name.
